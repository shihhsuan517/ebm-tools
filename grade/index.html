📖 Core GRADE 方法總覽

在開始逐步評分前，請先建立 GRADE 的整體框架概念。本頁摘錄自 Guyatt et al., BMJ 2025;389:e081903（Core GRADE 1）。

🗺 Core GRADE 整體流程（兩大部分）

左側：系統性回顧流程
① 以 PICO 格式定義回顧問題
② 識別並評估每個結局的重要性
③ 整合與總結證據
④ 評估每個結局的證據確定性
⑤ 呈現 Summary of Findings Table
右側：指引制定流程
① 考量效益大小、傷害與負擔
② 確認證據確定性
③ 納入病患價值觀與偏好
④ 考量費用、可行性、可接受性
⑤ 發出建議（方向 + 強度）
本工具聚焦於左側流程： 協助學生完成每個結局的 GRADE 確定性評分，最終產生 Summary of Findings Table。

⬤ 四個確定性等級

等級符號含義Plain Language起始研究設計
High ⬤⬤⬤⬤ 對真實效果非常有信心，估計值接近真實效果 治療具有效益 RCT
Moderate ⬤⬤⬤◯ 中等信心；真實效果可能接近估計值，但可能有實質差異 治療可能（likely）具有效益 —
Low ⬤⬤◯◯ 信心有限；真實效果可能與估計值有實質差異 治療或許（may）具有效益 NRSI / 觀察性研究
Very Low ⬤◯◯◯ 幾乎無信心；真實效果很可能與估計值存在實質差異 對治療效益極不確定 病例系列
⚠ 嚴格區分： "likely"（可能）對應 Moderate；"may"（或許）對應 Low。兩者在指引建議強度上意義截然不同，不可混用。

📉 降低確定性的五個領域 ＋ 📈 升級的兩個條件

方向領域說明最大降/升幅度

降級
（RCT 及 NRSI 均適用）		 ① 偏誤風險

② 不一致性

④ 不精確性

整體觀原則： 考量每個個別領域後，需後退一步進行整體評估。GRADE 是促進深思熟慮的判斷架構，而非機械式規則。

🎯 PICO 制定關鍵原則

結局重要性評分（9 分制）
7–9 分：關鍵結局（Critical）→ 最低確定性 = 整體確定性
4–6 分：重要但非關鍵（Important）
1–3 分：不重要（不納入評估）
亞組效應處理原則
• 需事先（a priori）指定，且≤3個
• 必須包含方向（如「老年人中效果更大」）
• 大量未指定方向的假設 → 損害可信度
相對效應 vs 絕對效應： 相對效應（RR/OR/HR）通常在亞組間穩定；絕對效應（風險差）對病患更重要。
同一 RR = 0.5：低風險族群絕對降低 1%；高風險族群絕對降低 20% → 可能導致不同的建議強度。

📋 建議方向與強度（供參考）

建議類型術語臨床意涵
強建議支持Recommend FOR大多數或幾乎所有知情病患都會選擇該介入措施
弱建議支持Suggest FOR大多數知情病患會選擇，但相當比例不會；強調共同決策
弱建議反對Suggest AGAINST大多數知情病患會拒絕，但相當比例不會
強建議反對Recommend AGAINST大多數或幾乎所有知情病患都會拒絕該介入措施
重要： 高確定性 ≠ 必然強建議。建議強度還取決於效益—傷害平衡、病患價值觀與偏好、費用與可行性。

第一步：研究設計起始確定性

Core GRADE 依照研究設計類型設定初始確定性等級。請選擇您文獻的主要研究設計。

🔬 研究設計類型

📘 Core GRADE 證據確定性四等級

等級標誌涵義Plain Language
High⬤⬤⬤⬤對真實效果非常有信心reduces / increases / has little to no effect
Moderate⬤⬤⬤◯中等信心；真實效果可能接近估計值probably reduces / increases
Low⬤⬤◯◯信心有限；真實效果可能有實質差異may reduce / increase
Very Low⬤◯◯◯幾乎無信心；效果非常不確定the effect is very uncertain
🔑 關鍵原則： RCT 從 High 開始 → 可能被扣分降級；NRSI 從 Low 開始 → 可能被扣分降至 Very Low，或因特殊條件升級至 Moderate

第二步：設定評估 Outcomes

GRADE 對每個 outcome 分別評估確定性。請輸入您文獻中要評分的 outcomes（至少 1 個，建議填寫 2-4 個）。

🎯 Outcome 重要性分類（9 分制）

分數分類說明
7–9關鍵 Critical必須納入評估；整體確定性由最低確定性的關鍵 outcome 決定
4–6重要 Important納入評估，但不決定整體確定性
1–3不重要通常不納入

➕ 新增 Outcome

💡 建議： 可直接輸入您的文獻 outcomes，例如：「角膜上皮癒合率」、「疼痛 VAS 分數」、「感染相關不良事件」

第三步：偏誤風險（Risk of Bias）

評估納入研究是否存在系統性偏差。偏誤風險是影響確定性最常見的降級原因。

📌 評估邏輯（依研究設計）

RCT 核心評估項目（ROBUST-RCT 優先）：
隨機序列生成、分配隱蔽、受試者盲化、醫護人員盲化、結果評估者盲化、遺漏數據
NRSI 核心評估（Newcastle-Ottawa / CLARITY）：
殘差干擾（Residual confounding）— NRSI 已因此從 Low 起始；若有額外偏誤，可繼續降至 Very Low
⚡ 偏誤方向考量： 若偏誤方向與研究結論相反（偏誤反強化結論），可不降級

第四步：不一致性（Inconsistency）

評估跨研究間的無法解釋的結果變異性。核心工具：Forest Plot 視覺判讀 + I² 統計量。

📊 判斷三大視覺標準

標準不一致性較低傾向降級
① 點估計差異各研究點估計相近（如 RR 0.71–0.76）研究間差異大（如 RR 0.44 vs. 1.25）
② CI 重疊程度各研究 CI 大部分重疊CI 完全不重疊，方向相反
③ 閾值同側絕大多數點估計在閾值同側相當比例分布在閾值兩側
I² 指引： ＜30% → 通常不降級 ｜ 30-60% → 結合視覺判斷 ｜ ＞60% → 傾向降級（但仍看閾值位置）
警告： I² 可能誤導！樣本大→假性高 I²；樣本小→假性低 I²。視覺判讀優先。
二元 vs 連續結果： 二元 → 以相對效果（RR/OR）判斷一致性；連續 → 以絕對效果（MD）判斷。
切勿以絕對效果評估二元結果的一致性 → 會產生假性不一致。

第五步：間接性（Indirectness）

評估 Target PICO 與現有研究 PICO 之間的差異。差異越大，證據越間接，確定性越低。
依據：Guyatt et al., Core GRADE 5, BMJ 2025;389:e083865

🎯 Indirectness 四大來源與降級可能性（Core GRADE 5, Table 2）

PICO 要素常見間接性原因降級可能性
P 族群 年齡差異（兒童↔成人↔老年）；病況隨時間改變（如 COVID-19 變異株）；共病症分布差異 通常較低
相對效果在族群間通常穩定；除非有機制上的理由認為會有差異
I 介入 藥物劑量 / 途徑 / 持續時間不同；High non-adherence；介入描述不足；技術演進（裝置）；試驗中允許轉換治療 中等可能性
取決於潛在生物學機制及 non-adherence 嚴重程度
C 比較 使用 placebo 而非 active comparator；使用次佳舊有替代方案；業界贊助試驗使用次佳 comparator 相當可能性
尤其在有效治療已存在、仍以 placebo 作對照時
O 結局 追蹤時間過短；使用 surrogate outcome 而非 patient-important outcome；測量方式不同（有症狀 vs 無症狀） 高可能性
Surrogate → 幾乎必降級；surrogate 可考慮降兩級
⚡ Surrogate Outcome 降級特別說明：
一般 indirectness 降一級；但若 surrogate 與 patient-important outcome 的關聯十分薄弱（如：透析病患的鈣磷代謝指標 vs. 心肌梗塞），可考慮降兩級。
骨密度 vs. 骨折 → 通常降一級 ｜ 血中生化指標 vs. 心血管死亡 → 可能降兩級
🔑 Indirectness vs. Inconsistency 判斷原則：
若同時有老年人與年輕人的資料 → 可「檢驗」效果差異 → Inconsistency 問題
若幾乎所有資料均來自年輕人，但需推論老年人 → 無法檢驗 → Indirectness 問題
📖 兩種遭遇 Indirectness 的情境（展開對照教材）
情境一：搜尋 Direct Evidence 過程中遭遇
找到的研究有些許 PICO 差異（族群年齡偏大、不同共病症）
→ 通常不需降級（共病症差異很少導致相對效果不同）
例外： High non-adherence、Surrogate outcome、次佳 comparator → 需降級
情境二：刻意搜尋 Indirect Evidence
Direct evidence 不存在 / 確定性極低時，改用其他族群或相關病況的資料
→ 幾乎必須降級（間接性是不可避免的）
案例： 兒科 pancreatitis → 以成人抗生素資料推論（降一級）
⚠ COVID-19 警示案例： Casirivimab / imdevimab 的 RCT 在 2020–2021 年對當時病毒有效，但 Omicron 變異株後 spike protein 改變 → 原始 RCT 對新族群變成 極度 indirect evidence → WHO 強烈建議反對繼續使用。
教學意涵： 族群「隨時間改變」也是 P 的 indirectness，需持續更新評估。

第六步：不精確性（Imprecision）

Core GRADE 第二篇核心流程：先選閾值 → 判斷 CI 與閾值關係 → 若效果大則評估 OIS 樣本數。每個 Outcome 獨立走完引導步驟後產生最終降級決策。

📖 展開教材參考表格（評估時對照用）

Step A：閾值選擇邏輯

MID 閾值（首選）Null 閾值（MID 不存在時）
核心問題效果是否達到臨床重要性？是否存在真實治療效果？
適用情境臨床指引；有病患觀點文獻；PRO 量表有 MID 數據MID 難以確立；系統回顧不納入 MID 判斷；二元結果基準風險極低
MID 來源病患回報結果研究、現有文獻、焦點小組、臨床共識統計學 null（RD = 0 或 MD = 0）

Step B–C：CI 與閾值關係 → 降級決策

情境描述Null 閾值MID 閾值
(a)點估計遠超 MID，CI 窄，未跨任一閾值✅ 不降級✅ 不降級
(b)點估計在 Null～MID 之間，CI 跨 MID 但未跨 Null✅ 不降級⬇ 降一級
(c)點估計近 Null，CI 寬，跨越 Null 及 MID 一側⬇ 降一級⬇ 降一級
(d)CI 極寬，同時涵蓋 MID for benefit 與 MID for harm（兩端）⬇⬇ 考慮降兩級⬇⬇ 考慮降兩級
降兩級觸發條件： ① CI 同時跨越 benefit 與 harm 兩端 MID ｜ ② 最適切的 plain language 應為 "may"（Low）而非 "likely"（Moderate）→ 任一條件成立即考慮降兩級

Step D–E：OIS 樣本數評估（效果大且 CI 未跨閾值時觸發）

觸發條件： CI 未跨越閾值（看似不需降級），但 RR reduction > 30–40%（二元）或效果量明顯大（連續）
原因： 早期小型研究的大效果往往無法後續複製（regression to the mean）
實際 N 與 OIS 比較降級決策
實際 N ≥ OIS（或連續結果每組 ≥ 400，共 ≥ 800）✅ 不降級
30% OIS < 實際 N < OIS⬇ 降一級
實際 N ≤ 30% OIS⬇⬇ 考慮降兩級
RR CI ratio ≥ 3 或 OR CI ratio ≥ 2.5⬇⬇ 建議進一步計算 OIS 確認
OIS 計算參數（二元）： α = 0.05，β = 0.20，control event rate，RRR 設 20–25%
連續結果簡化規則： 每組 ≥ 400 人（共 ≥ 800）→ 不降級
計算工具： OpenEpi.com

🔵 逐步引導評分

第七步：發表偏誤（Publication Bias）

陰性結果未發表 → 系統性高估效果。須主動懷疑，尤其是小型業界贊助研究。

🚨 高度懷疑發表偏誤的情境

最高風險：
全部研究均為小型 + 多數由業界贊助
案例：類黃酮痔瘡研究（11篇全數業界贊助）
可偵測方法：
漏斗圖視覺判讀（需≥10篇）
Egger's / Begg's 統計檢定
直接查閱 Registry / FDA 資料庫
Reboxetine 警示案例： 未發表研究揭露後，發現已發表數據高估效益達 115%。業界贊助的壓制是最危險的發表偏誤來源。

第八步：升級考量（Rating Up）

⚠ 重要： 升級 僅適用於觀察性研究（NRSI），RCT 不適用升級。

⬆ 兩大升級條件

① 大效果（Large Effect）
RR > 2 或 < 0.5 → 考慮升一級
RR > 5 或 < 0.2 → 考慮升兩級
前提：結果一致且無嚴重偏誤
② 劑量反應梯度（Dose-Response）
暴露量越大 → 效果越大 → 升一級
增加因果推論可信度
舊版第三升級條件「可預測偏誤方向」已從 Core GRADE 中移除，不再適用。
